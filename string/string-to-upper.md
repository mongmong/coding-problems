# String To Upper

## Problem

Input a string that contains arbitrary alphabets, and cast every lower case letter to upper case.

## Hints

Use ASCII table to translate lower case to upper case. 

For example: 


    lower_case = chr(ord('a') - 32)  # lower_case == 'A'


## Example

Input:

    Hello, world!

Output: 

    HELLO, WORLD!
